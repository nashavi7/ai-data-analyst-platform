# AI Data Analyst Platform

An AI-powered data analysis application that automates exploratory data analysis, feature insights, predictive modeling, and visualization. Built using Streamlit, this tool enables users to upload datasets and gain instant, intelligent insights without manual analysis.

---

## Features

- Automated data profiling (missing values, duplicates, statistics)
- KPI detection using statistical and heuristic methods
- Correlation analysis and feature relationships
- Outlier detection using IQR method
- AutoML modeling (classification & regression using Random Forest)
- Feature importance analysis
- AI-style narrative insights and recommendations
- Interactive visualizations using Plotly
- Clean, modern UI with Streamlit

---


## Architecture Overview
User Upload → Data Profiling → KPI Detection → Correlation Analysis → Outlier Detection → AutoML → Insight Generation → Visualization

---

## Tech Stack

- **Frontend/UI:** Streamlit  
- **Data Processing:** Pandas, NumPy  
- **Visualization:** Plotly  
- **Machine Learning:** Scikit-learn  
- **Statistics:** SciPy  
- **Modeling:** Random Forest (Classifier & Regressor)
